# 金某的个人主页

个人作品集 / 主页，修改自 <a href="https://github.com/RicLinccc/xbWeb">RicLinccc/xbWeb</a>。

## 技术栈

- 纯静态 HTML + CSS + JavaScript，无需构建工具
- [Tailwind CSS](https://tailwindcss.com/)（CDN）
- JetBrains Mono / Space Grotesk / Noto Serif SC 字体
- PWA 支持

## 特性

- 深色 / 浅色主题切换
- 响应式设计，移动端适配
- 打字机动效
- 平滑滚动

## 使用

直接用浏览器打开 `index.html` 即可，或部署到任意静态托管服务（GitHub Pages、Vercel 等）。
